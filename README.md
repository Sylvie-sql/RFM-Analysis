# Segmentation RFM
## Recency : Quand mon client a t il effectué sa dernière commande ?
## Frequency : A quelle fréquence mon client effectue t il des commandes ?
## Monetary Value : Combien de € mon client dépense t il par commande ?
RFM a beau être très efficace pour la plupart des cas business, il est possible d’en enrichir les indicateurs avec d’autres KPIs : 

1. Customer Lifetime Value (CLTV) 

2. Customer Satisfaction (CSAT) & Net Promoter Score (NPS) 

3. Probabilité de CHURN (attrition)
